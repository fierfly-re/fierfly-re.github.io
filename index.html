#Ubuntu22-04虚拟机和kali的搭建过程

Ubuntu22-04虚拟机的搭建
一、准备工作

  1. 软件下载
    1.1 下载VMware软件：前往VMware官网（https://www.vmware.com/）下载VMware Workstation Pro或VMware Player版本。
    1.2 下载Ubuntu 22.04镜像：前往Ubuntu官网（https://ubuntu.com/download/desktop）下载Ubuntu 22.04 ISO镜像文件。
  2. 硬件要求
    确保您的计算机硬件满足以下最低要求：
    • 处理器：支持64位
    • 内存：4GB及以上
    • 硬盘空间：至少40GB可用空间
    • 显卡：支持DirectX 11或更高版本
      二、安装VMware软件
  3. 双击下载的VMware安装包，开始安装过程。
  4. 根据提示完成安装，期间可自定义安装路径和其他设置。
  5. 安装完成后，启动VMware软件。
    三、创建Ubuntu 22.04虚拟机
  6. 打开VMware软件，点击“创建新虚拟机”。
  7. 选择“自定义(高级)”选项，然后点击“下一步”。
  8. 在“兼容性”界面，选择“Workstation 16.x”或更高版本，点击“下一步”。
  9. 在“安装操作系统”界面，选择“稍后安装操作系统”，点击“下一步”。
  10. 在“选择操作系统”界面，选择“Linux”版本为“Ubuntu 64位”，点击“下一步”。
  11. 命名虚拟机并选择安装位置，点击“下一步”。
  12. 在“处理器配置”界面，根据实际情况分配处理器核心数和每个核心的线程数，点击“下一步”。
  13. 在“内存”界面，分配虚拟机内存大小（建议至少分配2GB），点击“下一步”。
  14. 在“网络类型”界面，选择“使用桥接网络”，点击“下一步”。
  15. 在“选择I/O控制器类型”界面，默认即可，点击“下一步”。
  16. 在“选择磁盘类型”界面，默认即可，点击“下一步”。
  17. 在“选择磁盘”界面，选择“创建新虚拟磁盘”，然后设置磁盘大小（建议至少分配40GB），点击“下一步”。
  18. 在“指定磁盘文件”界面，默认即可，点击“下一步”。
  19. 查看虚拟机配置，确认无误后，点击“完成”。
    四、安装Ubuntu 22.04操作系统
  20. 选中创建好的虚拟机，点击“编辑虚拟机设置”。
  21. 在“硬件”选项卡中，找到“CD/DVD(IDE)”设备，选择“使用ISO文件”，然后浏览并选择下载的Ubuntu 22.04镜像文件。
  22. 点击“确定”，启动虚拟机。
  23. 等待虚拟机启动，进入Ubuntu安装界面。
  24. 选择“安装Ubuntu”选项，开始安装过程。
  25. 根据提示，完成语言、键盘布局、网络设置等配置。
  26. 在“安装类型”界面，选择“清除整个磁盘并安装Ubuntu”。
  27. 在“分区”界面，默认即可，点击“继续”。
  28. 设置用户名和密码，点击“继续”。
  29. 等待安装完成，重启虚拟机。
    五、配置Ubuntu 22.04虚拟机
  30. 登录Ubuntu系统，打开终端。
  31. 更新系统软件包：sudo apt update && sudo apt upgrade。
  32. 安装VMware Tools：在VMware菜单中选择“虚拟机”->“安装VMware Tools”。
  33. 解压VMware Tools安装包，进入解压后的目录，运行安装脚本：sudo ./vmware-install.pl。
  34. 根据提示完成安装，重启虚拟机。

kali的搭建
准备工作：

  • 在主机上安装VMware Workstation
  • 下载Kali Linux ISO镜像（可从https://www.kali.org/get-kali/获取）
  • 确保有足够的硬件资源（CPU、内存和磁盘空间）
    步骤1：下载Kali Linux ISO镜像
  1. 访问Kali Linux官方网站。
  2. 下载最新版的Kali Linux ISO文件（根据系统兼容性选择32位或64位）。
    步骤2：创建新的虚拟机
  3. 打开VMware Workstation。
  4. 点击“文件”并选择“新建虚拟机”，或直接点击“创建新虚拟机”按钮。
  5. 选择“自定义（高级）”并点击“下一步”。
  6. 选择“稍后安装操作系统”并点击“下一步”。
  7. 选择“Linux”作为客户机操作系统，并在下拉菜单中选择“Debian 10.x 64位”（或与您下载的Kali Linux版本相近的版本）。点击“下一步”。
  8. 为虚拟机命名并在选择存储位置。点击“下一步”。
  9. 选择要分配给虚拟机的处理器数量和核心数。点击“下一步”。
  10. 分配内存给虚拟机（至少推荐2GB内存给Kali Linux）。点击“下一步”。
  11. 选择网络连接类型。大多数用户选择“NAT”即可。点击“下一步”。
  12. 保持I/O控制器类型和SCSI控制器的默认设置，分别点击“下一步”。
  13. 选择“创建新虚拟磁盘”并设置磁盘大小（至少推荐20GB）。点击“下一步”。
  14. 除非有特殊需求，否则选择“将虚拟磁盘拆分为多个文件”。点击“下一步”。
  15. 查看虚拟机设置并点击“完成”。
    步骤3：配置虚拟机设置
  16. 在VMware Workstation库中右键点击新建的虚拟机，选择“设置”。
  17. 前往“CD/DVD（IDE）”部分。
  18. 选择“使用ISO文件”并浏览到您下载的Kali Linux ISO文件的位置。
  19. 点击“确定”关闭设置窗口。
    步骤4：安装Kali Linux
  20. 启动虚拟机以开始Kali Linux的安装过程。
  21. 在Kali Linux启动菜单中选择“图形安装”以进行图形化安装，或选择“安装”进行文本安装。
  22. 根据屏幕提示设置语言、国家和键盘布局。
  23. 配置网络设置，如果没有连接网络，可以跳过此步骤。
  24. 设置主机名和域名。
  25. 创建一个带有强密码的用户账户。
  26. 选择分区方法。除非有特殊需求，否则选择“向导 - 使用整个磁盘并设置LVM”。
  27. 确认更改并等待安装程序将文件复制到磁盘。
  28. 安装完成后，从虚拟机的CD/DVD设置中移除Kali Linux ISO文件，以防止再次启动。
  29. 重新启动虚拟机。
    步骤5：安装后操作
  30. 使用安装过程中创建的用户账户登录。

  32. 安装VMware Tools以获得更好的集成和性能：
    • 在VMware Workstation的“虚拟机”菜单中选择“安装VMware Tools”。
    • 挂载VMware Tools ISO并导航到挂载的目录。
    • 运行安装脚本：sudo ./vmware-install.pl并按照提示操作。
